# CloverDX Quick Start Guide - Sandbox Project

This repository contains the official sandbox project for the **[Quick Start Guide](https://academy.cloverdx.com/courses/quick-start-guide)** course at **CloverDX Academy**. 

This project is a hands-on companion designed to help you master the fundamentals of the CloverDX platform, including data transformation, automation, and orchestration.

---

## 🚀 Getting Started

To get the most out of this project, we recommend following the step-by-step instructions in the [CloverDX Academy Quick Start Guide](https://academy.cloverdx.com/courses/quick-start-guide).

### Prerequisites
* **CloverDX Designer:** Ensure you have the Designer installed.
* **CloverDX Server (Optional):** Required for the chapters covering automation, scheduling, and jobflow execution.

### Installation & Import
1. **Clone/Download:** Clone this repository or download the ZIP file and extract it.
2. **Import into Designer:**
   * Open CloverDX Designer.
   * Go to **File > Import...**
   * Select **General > Existing Projects into Workspace**.
   * Browse to the project folder and click **Finish**.

---

## 📚 Course Modules
The sandbox is organised to support the following learning path:

* **Introduction:** Navigating the CloverDX environment.
* **Transformations:** Creating your first `.grf` graphs to move and clean data.
* **Metadata:** Defining data structures for reusability.
* **Jobflows:** Orchestrating complex tasks and handling errors.
* **Automation:** Leveraging the Server for scheduling and event-based triggers.
* **Data Manager:** Simplifying data access for business users.
